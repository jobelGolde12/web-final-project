NARUTO WEBSITE

WEB SYSTEMS AND TECHNOLOGIES FINAL PROJECT

About This Project

This website is a static project showcasing some of the beloved characters from the Naruto universe.

It was created as a final project for our Web Systems and Technologies course. We aimed to demonstrate our understanding of HTML, CSS, and web development fundamentals.

We hope you enjoy exploring the information about Naruto, Minato, Sasuke, and Jiraiya. The images used are sourced from Google, and the links provided lead to the official Naruto website for further exploration.
